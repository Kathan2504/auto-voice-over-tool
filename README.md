## 📝 About auto-voice-over-tool

auto-voice-over-tool (AVOT) is a desktop app built to make voice-over easy for videos. It works on Windows and helps you add voice tracks automatically. You do not need any programming skills. The tool can download videos, extract audio, translate speech, and add new voice-overs.

It uses reliable tools like FFmpeg, Electron, and Whisper to handle multimedia processing and speech-to-text. This means AVOT will let you work with many video types and languages automatically.

## 🎯 What Does AVOT Do?

AVOT helps users to:

- Download videos from popular sources.
- Extract audio from video files.
- Transcribe speech into text.
- Translate the text into other languages.
- Generate new voice-over audio.
- Combine the new audio with the original video.
- Export finished videos ready to share.

It removes the manual work of recording and syncing voice-overs. Instead, it automates this in one program on your desktop.
